개인정보처리방침

최종 업데이트: 2026년 5월 2일

모도챗(이하 "서비스")은 사용자의 개인정보를 소중히 보호하며, 본 방침을 통해 수집·이용 항목과 사용자의 권리를 안내합니다.

1. 수집하는 개인정보

모도챗은 사용자의 이름, 이메일, 전화번호 등 식별 가능한 개인정보를 직접 수집하지 않습니다. 회원가입, 로그인, 결제 기능을 제공하지 않습니다.

사용자가 입력한 메시지는 응답 생성을 위해 즉시 외부 LLM API로 전달되며, IP 또는 식별 정보와 결합되어 서비스 서버에 영구 저장되지 않습니다.

2. 제3자 서비스를 통한 자동 수집

서비스 운영을 위해 다음 외부 서비스를 사용하며, 이들이 사용자의 IP 주소, 쿠키, 디바이스 정보, 방문 기록 등을 자동 수집할 수 있습니다:

서비스 목적 수집/처리 항목
Google Gemini API 응답 생성 (LLM) 사용자가 입력한 메시지 텍스트 (Google 정책에 따라 처리)
Cloudflare Workers AI 유사 질문 검색용 임베딩 생성 사용자가 입력한 메시지 텍스트
Google Analytics 4 방문 통계 분석 IP, 쿠키, 페이지 방문 기록, 디바이스 정보
Google Tag Manager 태그 관리 쿠키
Cloudflare Pages 호스팅·보안·레이트 리밋 IP, 서버 로그 (시간 단위 카운터, 24시간 후 자동 삭제)
Google AdSense 광고 게재 쿠키, 광고 식별자, 관심사 추정 정보

3. 응답 캐시에 관한 안내

비슷한 질문에 빠른 응답을 제공하기 위해, 익명화된 질문·응답 쌍이 서비스 내부 캐시에 저장됩니다. IP, 쿠키, 식별 정보는 함께 저장되지 않으며, 입력 메시지는 응답 검색·생성 외 다른 용도로 사용되지 않습니다.

4. 광고에 관한 안내

모도챗은 Google AdSense 또는 기타 광고 네트워크를 통해 광고를 게재할 수 있습니다. 해당 서비스는 사용자의 방문 기록을 기반으로 관심사 맞춤 광고를 제공합니다.

사용자는 Google 광고 설정 에서 광고 개인화를 거부할 수 있습니다.

5. 쿠키 사용 및 거부

본 서비스 및 제3자 서비스는 쿠키를 사용합니다. 사용자는 브라우저 설정에서 쿠키를 차단하거나 삭제할 수 있으며, 이 경우 일부 기능이 정상 동작하지 않을 수 있습니다.

6. 개인정보 보유·이용 기간

모도챗은 직접 개인정보를 보유하지 않습니다. IP 기반 레이트 리밋 카운터는 24시간 후 자동 삭제됩니다. 제3자 서비스의 데이터 보유 기간은 각 서비스의 정책을 따릅니다 (예: Google Analytics 기본 14개월).

7. 사용자의 권리

사용자는 본인의 개인정보에 대해 다음 권리를 행사할 수 있습니다:

  • 제3자 서비스에서의 데이터 수집 거부 (브라우저 쿠키 차단 등)
  • 광고 개인화 거부 (Google 광고 설정)
  • 본 방침에 관한 문의 및 의견 제출
